For commands that are long or complex, create a temporary script file (temp.py, temp.sh, temp.js, etc.) and run that instead of putting everything inline in bash.

**Why:** User preference for cleaner workflow and easier debugging.
**How to apply:** Whenever a bash command would be multi-line or complex, write it to a temp file first and execute that.
